なぜKGI・KPIが重要か？

～カスタマーマーケティングにおける羅針盤～

KGI・KPIとは？ (書籍のおさらい)

🎯KGI (Key Goal Indicator / 重要目標達成指標)

定義: 最終的な目標（ゴール）が達成されているかを計測する指標。

例: 売上高、利益率、市場シェア、LTV (顧客生涯価値)

一言: 「私たちが目指す山の頂上」

📜KPI (Key Performance Indicator / 重要業績評価指標)

定義: KGI達成に向けた各プロセスが適切に実行されているかを計測・評価するための中間指標。

例: Webサイトアクセス数、コンバージョン率、オンボーディング完了率、アクティブユーザー率

一言: 「頂上へ続く道のりを示す道しるべ」

事業目標とKGI・KPIの繋がり - 全体像を掴む

図解エリア：ピラミッド構造またはツリー構造
頂点: 会社のビジョン / 事業全体の目標
↓
事業部KGI (例: カスタマーサクセス部門KGI → 年間解約率●%未満)
↓
チームKGI / 主要KPI (例: オンボーディングチームKPI → 初回ログイン後7日以内の主要機能利用率●%)
↓
個人KPI / アクション指標 (例: オンボーディングコール実施件数)

ポイント： 「上位の目標からブレイクダウンして設定される」「全てのKPIはKGI達成に貢献する」

カスタマーマーケティングのKGI例 - 何を目指すか？

  • 👑LTV (顧客生涯価値) / CLV: 顧客が取引期間全体を通じて企業にもたらす総利益。カスタマーサクセスの究極目標の一つ。
  • 📉チャーンレート (解約率): 一定期間にどれだけの顧客がサービス利用を停止したかの割合。
  • 😊NPS® (ネットプロモータースコア): 顧客ロイヤルティを測る指標。「推奨者」-「批判者」の割合。
  • 🔄リテンションレート (顧客維持率): 既存顧客が継続してサービスを利用している割合。
  • 💰アップセル/クロスセル金額・件数: 既存顧客への追加販売による売上。

これらはあくまで一例です。事業フェーズや戦略によって最適なKGIは異なります。

カスタマーマーケティングのKPI例 - KGI達成への道筋

オンボーディング関連：

  • オンボーディング完了率
  • 初期設定完了までの平均日数
  • 初回ログイン後●日以内の主要機能利用率

アダプション (利用定着) 関連：

  • アクティブユーザー率 (MAU/WAU/DAU)
  • 主要機能の利用頻度・利用率
  • セッション時間

エンゲージメント関連：

  • コミュニティ参加率・投稿数
  • セミナー/ウェビナー参加率
  • CSAT (顧客満足度スコア)

エクスパンション (顧客価値向上) 関連：

  • アップセル/クロスセルのリード数・商談化率
  • 特定プランへのアップグレード率

KPIはKGI達成への具体的な行動を促す「道しるべ」です。

先行指標と遅行指標 - カスタマーサクセスの鍵

🏁遅行指標 (Lagging Indicator)

結果として現れる指標。過去の活動の成果。

例: 売上、利益、解約率、LTV

特徴: 把握はしやすいが、直接コントロールしにくい。

🚀先行指標 (Leading Indicator)

遅行指標に先行して変化する指標。未来の結果を予測する。

例: アクティブユーザー数、機能利用率、エンゲージメントスコア、オンボーディング完了率

特徴: 直接的なアクションで改善しやすく、未来の結果を変える力がある。

図解エリア： 遅行指標「結果」 vs 先行指標「原因/活動」のシンプルな対比図

カスタマーマーケティングのポイント： 「解約率（遅行指標）が悪化する前に、先行指標の変化を捉え、先手を打つことが重要！」

なぜKPIは行動に繋がらない？ - よくある落とし穴

  • ❓目的が不明確: KGIとの繋がりが見えず、何のためにやっているかわからない。
  • ✖️測定できない/しにくい: データが取れない、または集計に手間がかかりすぎる。
  • 👤担当者がいない/曖昧: 誰がそのKPIに責任を持つのか決まっていない。
  • 🚫アクションに繋がらない: KPIが上下しても、具体的に何をすれば良いかわからない。 [*]
  • 📚KPIが多すぎる: 何を優先すべきかわからなくなる。
  • 🗓️見直しがされない: 設定しっぱなしで、状況変化に対応できていない。

皆さんのチームでは、当てはまるものはありますか？

良いKPIの条件 - SMARTの法則

S - Specific (具体的、明確であるか): 誰が読んでも同じ解釈ができるか？

例：「顧客満足度を上げる」→「●●機能利用者のCSATを前期比5ポイント向上させる」

M - Measurable (測定可能であるか): 定量的に測定できるか？進捗がわかるか？

例：「多くの人に使ってもらう」→「MAUを現在の1000から1200に増やす」

A - Achievable (達成可能であるか): 現実的に達成できる範囲か？現実離れしていないか？

例：「解約率を0%にする」→「解約率を現在の3%から2.5%に低減する」

R - Relevant (関連性があるか): KGI達成や事業目標に貢献するか？

例：(カスタマーサクセスチームが)「新規リード獲得数10%増」(←関連性が薄い可能性)

T - Time-bound (期限が明確であるか): いつまでに達成するのか？

例：「機能利用率を上げる」→「次四半期末までに主要機能Aの利用率を15%向上させる」

ワーク1：私たちのKGI・KPIを洗い出す

～自業務に落とし込み、具体的な一歩を踏み出す～

所要時間: 個人ワーク 15分 / グループ共有・FB 20分

ワーク1の目的と進め方

🎯目的：

  • 担当業務におけるKGI・KPI候補を具体的に言語化する。
  • SMARTの法則を意識してKPIの質を高める。
  • チームメンバーと共有し、多角的な視点を得る。

🛠️進め方：

  1. 個人ワーク (15分): 配布するワークシートに、自身の業務に関連するKGI・KPI候補を記入。
  2. グループワーク (20分): 3-4名のグループに分かれ、各自のKGI・KPIを共有。相互にフィードバックを行う。

配布物: KGI・KPI洗い出しワークシート (Excel)

個人ワークのポイント (15分)

  1. まずはKGIから: あなたのチーム/担当業務の「最終目標」は何ですか？ (ワークシートのKGI欄に記入)
  2. KPIを複数洗い出す: そのKGIを達成するために、どんな「道しるべ」が必要ですか？思いつくままに複数書き出してみましょう。（質より量！）
  3. SMARTチェック: 書き出したKPIをSMARTの観点で見直してみましょう。「具体的か？」「測れるか？」「達成できそうか？」「KGIと関連しているか？」「いつまでに？」
  4. [*]自業務への落とし込み: 普段の業務シーンを具体的に想像しながら、「このKPIが改善したら、どんな良いことがあるだろう？」「このKPIを良くするために、明日から何ができるだろう？」と考えてみましょう。
  5. [*]顧客セグメントも意識 (できれば): 担当している顧客セグメントによって、重視するKPIは変わりますか？

完璧を目指さなくてOK！まずは書き出すことが大切です。

グループワークのポイント (20分)

役割: 各グループでタイムキーパーを決めましょう。

共有: 一人ずつ、個人ワークで作成したKGI・KPI（特に自信のあるもの、悩んでいるもの）を発表。(1人あたり発表3分、FB2分程度)

フィードバックの観点 (建設的に！):

  • 👍そのKPIの良い点、共感できる点は？
  • ❓もっと具体的にするとしたら？ (SMARTの観点)
  • 💡他にこんなKPIも考えられないか？
  • 🔗そのKPIで本当にKGI達成に繋がるか？
  • 🏃そのKPIを改善するために、どんなアクションが考えられるか？

講師が各グループを回り、アドバイスや質問対応をします。

Day1の振り返り - 学びのポイント

  • KGIは「山の頂上」、KPIは「道しるべ」。
  • カスタマーマーケティングでは「先行指標」を意識することが重要。
  • 良いKPIは「SMART」であり、具体的な「アクション」に繋がる。
  • 自業務に落とし込み、チームで共有することでKPIは磨かれる。
